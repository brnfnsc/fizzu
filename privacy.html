Política de Privacidade — Fizzu

Última atualização: 2026

O Fizzu respeita a privacidade dos usuários e protege seus dados pessoais. Esta política descreve como as informações são coletadas e utilizadas.

Dados coletados

  • Informações de cadastro
  • Dados de agenda e compromissos
  • Dados de pacientes inseridos pelo profissional
  • Dados de autenticação Google quando autorizado pelo usuário

Uso da Conta Google

Quando autorizado, o Fizzu pode acessar dados de agenda do Google Calendar exclusivamente para sincronizar compromissos profissionais dentro do app. Nenhum dado é compartilhado com terceiros.

Uso das informações

  • Operação das funcionalidades do app
  • Sincronização de agenda
  • Organização de atendimentos
  • Controle financeiro do usuário

Compartilhamento

Não vendemos nem compartilhamos dados pessoais com terceiros.

Armazenamento

Os dados são armazenados em infraestrutura segura com controle de acesso.
