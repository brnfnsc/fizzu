Política de Privacidade — Fizzu

Última atualização: 2026

O Fizzu protege a privacidade dos usuários e trata dados com responsabilidade e segurança.

Dados coletados

  • Dados de cadastro do profissional
  • Informações de agenda
  • Dados de pacientes inseridos pelo usuário
  • Dados de autenticação Google quando autorizado

Integração com Google

Quando autorizado, o Fizzu acessa dados do Google Agenda apenas para sincronização de compromissos. Nenhuma informação é compartilhada com terceiros.

Uso das informações

  • Operação das funcionalidades do app
  • Organização de atendimentos
  • Controle financeiro

Compartilhamento

Não vendemos nem compartilhamos dados pessoais.
